# lab1

Сеть состоит из пяти слоёв: 1 Flatten  и 4 Dense. В первых трёх слоях Dense используется активация relu, в последнем linear. В первом слое Dense используется 128 нейронов, во втором - 64, в третьем - 64, в четвёртом - 10. В обучении используются оптимизатор adam, функция потерь SparseCategoricalCrossentropy и метрика acc. Всего производится 13 эпох обучения.

Итоговая точность на тестовых данных 0.7069
